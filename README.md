# Vant.Ai 🚀 - Inteligência Comercial para Vendedores

**Vant.Ai** é uma ferramenta de apoio à prospecção e vendas, desenvolvida como projeto acadêmico para o curso de **Desenvolvimento Web** na Faculdade Nova Roma.

O sistema ajuda vendedores e empreendedores a identificar leads estratégicos e gerar "playbooks" (roteiros de abordagem) baseados em metodologias consagradas como **BANT** e **SPIN Selling**.

---

## 🎯 Objetivo do Projeto
Facilitar a jornada do vendedor, transformando dados brutos de empresas em estratégias de abordagem personalizadas, aumentando a taxa de conversão no primeiro contato.

## 🛠️ Tecnologias Utilizadas
Este projeto foi desenvolvido seguindo requisitos acadêmicos estritos de utilização de tecnologias fundamentais de web:
* **HTML5:** Estruturação semântica de todas as páginas.
* **Atributos Nativos:** Utilização de recursos nativos do HTML para formulários e validações.

> **Nota:** Conforme os requisitos, este projeto não utiliza CSS externo ou JavaScript para manter o foco na estrutura pura do HTML.

---

## 📂 Estrutura do Repositório
Conforme as normas estabelecidas:

  |-- index.html          # Landing Page (Página Inicial)
  |-- login.html          # Tela de Acesso
  |-- cadastro.html       # Formulário de Novos Usuários
  |-- lista-leads.html          # Gerador de Leads (Tabela de prospecção)
  |-- configurar-playbook.html     # Definição de estratégia (BANT/SPIN)
  |-- resultado- final.html      # Playbook final gerado
  |-- /imagens            # Logo e ativos visuais da marca

---

## 🏁 Como visualizar o projeto
1. Faça o clone do repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox).
3. Navegue entre as telas utilizando os botões e links do sistema.
